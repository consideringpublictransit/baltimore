The CSV files have the following format:

start TAZ, number of destinations, time limit, number of job TAZs, job average, job SD, number of food TAZs, food average, food SD

start TAZ: where we started
number of destinations: how many places can be reached under the time limit
number of job TAZs: how many TAZs that can be reached had non-zero job density
job average: average job density of all reachable (including zero desnity)
job sd: standard deviation of job average
number of food TAZs: see jobs
food average: see jobs
food sd: see jobs
